Users manual
============

Create a new content-element => plugin “Filelist”.

#. Folder: Select the folder, which contains the files.

#. Sort by: Choose the way, how the files should be sorted

#. Order: Ascending / Descending

#. File extensions: Comma separated list of file extensions, for example "pdf,jpg". Leave this field empty, if you want to list all files in the directory.

#. Cut filename: Cut the filenames after a number of letters

**Note:** Every Link to a file has a Css-class named like the file-extension, for example class=”pdf”.
